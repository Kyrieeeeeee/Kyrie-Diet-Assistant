# 🍱 饮食助手 (DietPlan) - 智能健美食谱生成器

饮食助手是一款专为健身人群、减脂/增肌爱好者设计的“高精度”微信小程序。它不仅能计算基础的 TDEE 和宏量营养素（Macros），还能根据**训练时段（Nutrient Timing）**、**有氧消耗**以及**现实生活中的复合食物（如外卖盒饭）**，动态、弹性地为你生成每一餐的精确克数配额。

## ✨ 核心功能与硬核亮点

### 1. 动态热量补偿系统 (Dynamic Calorie Compensation)
* **力量训练补偿**：根据性别自动判定力量训练的额外消耗。
* **有氧精细计算**：内置 9 种有氧项目（如平地走、爬坡、跑步、单车等），基于运动强度（METs）、用户实时体重和运动时长，精准计算当日有氧消耗并计入 TDEE。

### 2. 智能营养时机分配 (Nutrient Timing)
* 抛弃死板的“三分法”，系统会根据你的**训练时段**（晨练、上午、下午、傍晚、夜间）自动调整分餐策略。
* **核心练后餐（Post-workout Meal）**：自动将全天最大比例的碳水化合物和蛋白质锚定在训练后的一餐，最大化糖原储备与肌肉合成。

### 3. 🍱 复合食物弹性分配算法 (Elastic Combo Allocation) —— 🔥 核心技术
真实的打工人很难做到每顿都自己做饭。本系统独创了**“优先扣除 + 弹性瓜分”**算法：
* **复合库组装**：允许用户从基础食材库中拼装复合食物（如：减脂轻食 = 150g杂粮饭 + 100g鸡胸肉）。
* **优先锁定**：如果在某餐（如午餐）选择了复合食物，系统会优先扣除该食物的绝对宏量营养素。
* **完美按比缩放**：无论复合食物吃多吃少，系统都会自动计算全天**剩余配额**，并严格按照原有的饮食节奏（比例）完美摊派给早、晚等其他餐次，绝不破坏全天总目标！

### 4. 强大的本地食材库 (Food Library)
* 支持单营养来源（碳水主食、蛋白质来源）的手动录入与营养率换算。
* 支持自定义生活化单位（如：1 盒 = 100g，1 块 = 50g），告别反人类的纯克数称重。
* 支持复合食物的二次编辑，修改后全链路数据自动回填更新。

## 🛠️ 技术栈

* **框架**: 微信小程序原生框架 (WXML, WXSS, JavaScript)
* **存储**: 原生 `wx.getStorageSync` 本地缓存（无需后端数据库，保护用户隐私，极速响应）
* **UI风格**: Apple iOS Human Interface 极简风格（大圆角、毛玻璃模糊、高对比度强调色）

## 🚀 本地运行指南

1. 下载或克隆本仓库到本地。
2. 打开 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。
3. 点击 **+** 导入项目，选择本项目的根目录。
4. AppID 可以选择 **测试号**。
5. 编译运行即可在模拟器中体验（推荐使用 iPhone 12/13/14 Pro 比例进行预览）。
